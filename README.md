## Описание проекта

В рамках практической работы реализована двумерная среда управления автономным дроном в постановке марковского процесса принятия решений (MDP).

В проекте реализованы и сравнены три табличных агента обучения с подкреплением:

- рациональный агент (**Q-learning**);
- поведенческий агент (**Prospect Theory Q-learning**);
- риск-чувствительный агент (**Risk-Sensitive Q-learning**).

Проведены обучение и тестирование агентов, сравнение их эффективности по основным метрикам, а также визуализация результатов экспериментов.
